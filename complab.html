Public Computer Lab

The library offers a free public computer lab to both patrons and non patrons. The computer lab can be accessed
without a library card, but users must visit the Circulation Desk for an assigned computer pass. The public
computers gives patrons internet and office access, as well as printing.

Guidelines

  • Users must obtain a public computer pass
  • Users must adhere to the Acceptable Internet Use policy
  • Users are responsible for their personal information and should log out
    of all applications and avoid saving sensitive information to the desktop.
  • Users should not install software or download programs to library computers

Printing From Public Computers

  • Users are responsible for all printed pages
  • Users can visit the Circulation Desk to pay for and obtain their pages
  • Users can request specific computers to scan physical documents to the desktop
  • Users are responsible for their information. Please delete your documents before
    returning your pass.
  • Fees for printing can be found here

Public Wi-Fi Networks

The library offers two free internet networks to the public. Both patrons and non-patrons can use these networks.

Private Network

  • Users will need an internet pass and password to connect
  • Users will have printer access
  • Network can only be accessed from inside the building

Public Network

  • Users will need a one-time pass
  • Password does not change
  • Can be accessed from the parking lot and Abbitt Park
  • Does grant printer access
  • Can be acessed after business hours
